# Request_Creator
Python script with a weekly random schedual that sends http requests to a specific IP ( AWS Project )
_______

# Needed Libraries:

- Python3
- requests
- tornado import httpclient, ioloop
- colorama import init,Fore, Style
- time
- datetime
- schedule
- (Windows: I use Cygwin terminal and VS.Code Editor )

# Usage:

- Set Number of requests
- Edit your schedual (period of time to when the requests will be sent, read **Schedual** library docs first)
- Set destination IP
- Run the script :)
